## 📖 Overview

This project was developed as part of an Object-Oriented Programming (OOP) course to demonstrate the practical application of fundamental OOP principles in software development.

The project focuses on designing a modular, maintainable, and scalable system using classes, objects, inheritance, polymorphism, abstraction, and encapsulation. It serves as an educational example of how object-oriented design can be applied to solve real-world problems efficiently.

## 🏗 OOP Concepts Implemented

### Encapsulation

Data members are protected within classes using access modifiers and getter/setter methods.

### Inheritance

Child classes inherit common properties and methods from parent classes to promote code reusability.

### Polymorphism

Methods are overridden and dynamically invoked based on object type.

### Abstraction

Abstract classes and interfaces are used to hide implementation details and expose only essential functionalities.
